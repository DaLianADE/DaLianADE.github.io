# DaLian Analysis and Differential Equations

Official website source for the DaLian Analysis and Differential Equations Group at the School of Mathematical Sciences, Dalian University of Technology.

## Website

GitHub Pages publishes the `main` branch at <https://dalianade.github.io/>.

## Recruitment PDF

Edit `recruitment/DLADE-open-positions.tex` and push the change to `main`. The `Build recruitment PDF from TeX` workflow compiles the XeLaTeX source and updates `assets/DLADE-open-positions.pdf`; the website download buttons always point to that generated PDF.

The TeX source is kept in the repository for maintainers, but the public website exposes only the PDF download.
